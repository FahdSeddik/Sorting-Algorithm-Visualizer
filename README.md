# Sorting-Algorithm-Visualizer  
Items that are being switched/check are lit up either green or red

R - Reset array (randomize)  
SPACE - Start sorting with chosen sorting algorithm  

Implemented sorting algorithms:  
- Insertion Sort  
- Quick Sort  
- Selection Sort
